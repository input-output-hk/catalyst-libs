# Glossary

<!-- See: https://docs.arc42.org/section-12/ -->

| Term        | Definition        |
|-------------|-------------------|
| **Arc42** | Architecture documentation standard used for organizing system architecture documentation |
| **CBOR** | Concise Binary Object Representation - a binary data format used for encoding data structures |
| **CDDL** | Concise Data Definition Language - a language for defining data structures, used for CBOR schemas |
| **CIP** | Cardano Improvement Proposal - standards and specifications for the Cardano ecosystem |
| **CIP-30** | Wallet DAppBridge specification for Cardano wallet integration |
| **CIP-36** | Voting registration specification for Cardano governance |
| **CIP-95** | Message signing specification for Cardano |
| **C509** | Certificate format used in Catalyst for role-based access control |
| **COSE** | CBOR Object Signing and Encryption - format used for Catalyst signed documents |
| **Catalyst** | Cardano's community funding and governance program |
| **Catalyst ID URI** | URI scheme (id.catalyst://) for unambiguously identifying RBAC keys and registrations |
| **Catalyst Signed Document** | Document format with cryptographic signatures, versioning, and collaboration features |
| **Chain Follower** | Component that reads and follows Cardano blockchain updates |
| **cat-gateway** | Catalyst Data Gateway - backend service that uses catalyst-libs crates |
| **Catalyst Voices** | Flutter application for Catalyst governance and voting |
| **Event DB** | PostgreSQL database used by cat-gateway for event data |
| **Index DB** | ScyllaDB database used by cat-gateway for indexing and queries |
| **FFI** | Foreign Function Interface - mechanism for calling Rust code from other languages |
| **Hermes** | Runtime extension concept (may be outdated - see chain follower instances documentation) |
| **IPFS** | InterPlanetary File System - decentralized storage network |
| **Mithril** | Cardano protocol for efficient blockchain snapshots |
| **Monorepo** | Single repository containing multiple related projects or packages |
| **Node-to-Node Protocol** | Cardano protocol for direct communication with Cardano nodes |
| **RBAC** | Role-Based Access Control - system for managing permissions based on roles |
| **RBAC Registration** | On-chain registration of RBAC keys and roles on Cardano blockchain |
| **Vote Transaction** | Cardano transaction containing voting data (v1 or v2 format) |
| **WebAssembly (WASM)** | Binary instruction format for web applications, supported by some catalyst-libs crates |
